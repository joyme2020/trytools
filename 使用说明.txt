Unix时间戳在线转换工具 - 使用说明
=====================================

这个工具用来在"Unix时间戳"和"年月日时分秒"之间互相转换。
完全免费，无需注册，打开浏览器就能用。


一、什么是时间戳？
------------------

时间戳就是一个数字，表示从1970年1月1日0点开始，到某个时刻一共过了多少秒。

比如：
  1715385600  =  2024年5月11日 0点（UTC时间）

编程、写代码、调接口时经常用到。


二、页面布局
------------------

打开页面后，从上到下是：

  1. 实时时间戳  - 显示当前这一刻的时间戳，每秒自动刷新
  2. 时间戳转日期 - 输入时间戳，看它对应什么日期
  3. 日期转时间戳 - 输入日期，看它的时间戳是多少
  4. 时区参考表  - 12个主要时区的当前时间


三、怎么用：时间戳转日期
------------------

  第一步：在"Timestamp to Human-Readable Date"下面的输入框里，输入一个时间戳数字

      比如输入：1715385600

  第二步：右边会自动出现结果：
      - Unix (seconds)    1715385600
      - Unix (millisec)   1715385600000
      - ISO 8601          2024-05-11T00:00:00
      - Formatted         Sat, May 11, 2024, 08:00:00
      - RFC 2822          Sat, 11 May 2024 00:00:00 GMT
      - UTC               2024-05-11T00:00:00.000Z

  第三步：点任意一行右边的 Copy 按钮，把结果复制到剪贴板

  提示：
  - 数字小于1千亿，自动当成"秒"处理
  - 数字大于1千亿，自动当成"毫秒"处理
  - 也可以手动选 "Seconds" 或 "Milliseconds"
  - 数字里可以有逗号、空格，比如 1,715,385,600


四、怎么用：日期转时间戳
------------------

  第一步：在"Date to Timestamp"下面的输入框里，输入一个日期

      比如输入：2024-05-11 14:30

  第二步：右边自动出现：
      - Unix (seconds)    1715434200
      - Unix (millisec)   1715434200000
      - UTC               2024-05-11T06:30:00.000Z

  第三步：如果想转换其他时区的时间，在输入框右边的下拉菜单里选择时区

      比如选 "New York (EST)"，14:30就会被当成纽约时间来算


五、自然语言输入（非常方便！）
------------------

不需要记任何格式，直接打字就行，工具会自动理解：

  输入 now              得到 现在这一刻
  输入 today            得到 今天凌晨0点
  输入 tomorrow         得到 明天凌晨0点
  输入 yesterday        得到 昨天凌晨0点
  输入 3 days ago       得到 3天前现在这个时间
  输入 2 hours later    得到 2小时后的时间
  输入 next Friday      得到 下周五凌晨0点
  输入 next Monday at 3pm  得到 下周一下午3点
  输入 last Sunday      得到 上周日凌晨0点
  输入 3pm              得到 今天下午3点
  输入 noon             得到 今天中午12点
  输入 midnight         得到 今天凌晨0点
  输入 14:30            得到 今天下午2点半

以上所有表达式都可以配合时区选择器使用！


六、快捷操作
------------------

  Ctrl + K       键盘快捷键，光标直接跳到输入框
  Copy 按钮      每个结果旁边都有，点一下就能复制
  网址参数       在网址后面加 ?ts=数字，直接打开并转换
                比如：https://你的网址/?ts=1715385600


七、时区怎么选
------------------

两个转换区域都有时区下拉菜单，选项包括：

  本地时间    你电脑当前所在的时区
  UTC         世界标准时间
  上海 CST    北京时间
  纽约 EST    美国东部时间
  洛杉矶 PST  美国西部时间
  伦敦 GMT    英国时间
  东京 JST    日本时间
  新加坡 SGT  新加坡时间
  柏林 CET    德国时间

页面下方还有一个12时区实时对照表，一眼就能看到各地现在几点。


八、部署你自己的版本
------------------

这个工具就是一个 index.html 文件，可以部署到任何地方：

  Vercel（推荐）：
    打开命令行，进入文件夹，运行：npx vercel --prod

  GitHub Pages / Netlify / 任何静态文件托管：
    直接把 index.html 上传上去就行

  放到你自己的网站里：
    把 index.html 复制到你网站目录下，改个名字就行


九、接广告（AdSense）
------------------

文件里已经预留了两个广告位，找到 index.html 里面的：

  ADSENSE_SLOT_TOP    替换为你的顶部广告代码
  ADSENSE_SLOT_BOTTOM 替换为你的底部广告代码


十、隐私说明
------------------

所有计算都在你的浏览器里完成，不会上传任何数据到服务器。
你输入的内容、转换的结果，都不会被记录。
